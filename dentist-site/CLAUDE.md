## Суть проекта
Сайт-воронка — не визитка, а машина сбора базы пациентов.
Пациент приходит тёплым после приёма → оставляет отзыв через TG-бота →
читает статьи → уходит в TG-канал врача. База остаётся с врачом при смене клиники.

## Бизнес-модель масштабирования
- Цена сайта: 19 900 (первые 10) → 25 000
- Подписка поддержка: 3 000/мес
- Цель: 15 сайтов/мес = 1 млн на 12-й месяц

## Психологическая модель — "Тёплый коридор"
Не убеждаем, не давим. Один путь без развилок.
Контакт берётся как побочный результат хорошего опыта.

## Структура сайта (6 экранов — КОНСТАНТА)
1. Обложка — фото, имя, специализация, кнопка "Оставить отзыв" → TG-бот
2. Триггерные ситуации — карточки болей пациента → статьи в TG-канале
3. Рекомендации после лечения — кнопка → TG-бот отправляет памятку
4. Отзывы — реальные, из TG-бота
5. CTA — кнопка в TG-канал врача
6. Футер — адрес, телефон

## Дизайн (ПЕРЕМЕННАЯ — меняется под каждого врача)
Темы в: src/css/themes.css
- Тема 1: Тёмно-зелёная (Артур — default)
- Тема 2: Тёмно-синяя
- Тема 3: Тёплая бежевая
Иконки: SVG, никаких эмодзи
Блоки специализаций: Лечение · Удаление · Имплантация · Протезирование

## Правила текстов (ВАЖНО)
- От первого лица, как живой врач говорит в коридоре
- Коротко: ситуация (1 строка) + что сделаю (1-2 строки)
- Про пациента, не про врача
- ЗАПРЕЩЕНО: "разбираю схемы", "объясняю когда цена оправдана", любой копирайтерский официоз

## Технологии
- Сайт: HTML + CSS + vanilla JS → GitHub Pages (бесплатно)
- Бот: Node.js + Telegraf.js → VPS (150 руб/мес)
- Контакты: Google Sheets API (бесплатно до 500 пациентов)
- Деплой сайта: git push → GitHub Pages автообновление за 1-2 мин
