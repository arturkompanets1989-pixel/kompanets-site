# Папка images — фото врача

## Что сюда положить
Своё фото: `photo.jpg` или `photo.png`

## Требования к фото
- Вертикальное (портрет), пропорция 3:4
- Минимум 600×800 пикселей
- Фон нейтральный или размытый
- Лицо хорошо видно, улыбка
- Формат: JPG (размер до 300 КБ) или PNG

## Как подключить в сайт
Открой `src/index.html`, найди строку:

```html
<!-- <img src="images/photo.jpg" alt="Артур Компанец" class="hero-photo"> -->
```

Убери комментарий (удали `<!--` и `-->`), и удали строку с заглушкой:
```html
<div class="hero-photo-placeholder">...</div>
```

Итого должно остаться:
```html
<img src="images/photo.jpg" alt="Артур Компанец" class="hero-photo">
```
