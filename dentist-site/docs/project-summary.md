# Суммари проекта

## Кто
Артур Компанец, стоматолог (терапевт / ортопед / хирург), 16 лет опыта.
Санкт-Петербург, ул. Вавиловых 7 к4 | +7 895 169-08-99

## Что
Сайт-воронка для стоматолога — личный сайт врача заточенный не на красоту,
а на сбор собственной базы пациентов.

## Механика
1. Врач после приёма говорит скрипт → пациент ищет сайт
2. Пациент читает триггерные статьи → уходит в TG-канал врача
3. Оставляет отзыв через TG-бота → бот берёт контакт
4. Получает памятку после лечения → 2-й касание с контактом
5. База остаётся с врачом при любой смене клиники

## Бизнес-модель
- Создание сайта: 19 900 (первые 10 клиентов) → 25 000
- Поддержка: 3 000/мес
- Целевой масштаб: 15 сайтов/мес = 1 млн на 12-й месяц
- Аудитория: 660 TG + 1300 Instagram

## Психологическая модель — "Тёплый коридор"
Пациент приходит уже тёплым после приёма.
Не убеждаем, не давим. Один путь без развилок.
Контакт берётся как побочный результат хорошего опыта.
