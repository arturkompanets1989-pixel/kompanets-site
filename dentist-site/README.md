# Сайт-воронка стоматолога

Личный сайт врача заточенный на сбор базы пациентов через TG-бота.

## Быстрый старт

```bash
# Открыть сайт локально
open src/index.html   # Mac
# или просто двойной клик на файле в проводнике
```

## Структура

```
dentist-site/
├── src/              → Сайт (HTML + CSS + JS)
├── bot/              → TG-бот (Node.js)
├── content/          → Тексты (триггеры, памятки, статьи)
├── docs/             → Правила дизайна и текстов
├── deploy/           → Инструкция деплоя
├── CLAUDE.md         → Память проекта (читает Claude Code)
└── .env.example      → Шаблон переменных для бота
```

## Смена цветовой темы
Открой `src/css/themes.css` — там 3 темы, раскомментируй нужную.

## Запуск бота
```bash
cd bot
npm install
cp ../.env.example .env   # заполни токены
npm start
```
